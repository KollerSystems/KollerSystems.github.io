Koller Dokumentáció

Bejelentkezés (Refresh Token segítségével)

Kérés a szervertől

POST/oauth/token

HEADERContent-Type = application/json vagy application/x-www-form-urlencoded

application/json

{
	"grant_type": "refresh_token",
	"refresh_token": "xxxxxxxxxx"
}

application/x-www-form-urlencoded

grant_type=refresh_token&refresh_token=xxxxxxxxxx

Válasz a szervertől

Sikeres bejelentkezés: (200 OK)

{
	"access_token": "xxxxxxxxxx",
	"token_type": "Bearer",
	"expires_in": 86400,
	"refresh_token": "xxxxxxxxxx"
}

Hiányzik a típus paraméter vagy helytelen: (400 Bad Request)

{
	"error": "Grant type parameter missing or type not allowed!"
}

Hiányzik a refresh token paraméter: (400 Bad Request)

{
	"error": "A valid refresh token is required!"
}

Helytelen refresh token: (400 Bad Request)

{
	"error": "Invalid refresh token!"
}

Helytelen Content-Type: (400 Bad Request)

{
	"error": "Invalid Content-Type value or header missing!"
}

Helytelen body: (400 Bad Request)

{
	"error": "Invalid data in body!"
}

Helytelen URL: (404 Not Found)

{
	"error": "Page not found!"
}
