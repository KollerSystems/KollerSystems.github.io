Koller Dokumentáció

Bejelentkezés (Felhasználónév és jelszóval)

Kérés a szervertől

POST/oauth/token

HEADERContent-Type = application/json vagy application/x-www-form-urlencoded

application/json

{
	"grant_type": "password",
	"username": "xxxxxxxxxx",
	"password": "xxxxxxxxxx"
}

application/x-www-form-urlencoded

grant_type=password&username=xxxxxxxxxx&password=xxxxxxxxxx

Válasz a szervertől

Sikeres bejelentkezés: (200 OK)

{
	"access_token": "xxxxxxxxxx",
	"token_type": "Bearer",
	"expires_in": 86400,
	"refresh_token": "xxxxxxxxxx"
}

Hiányzik a típus paraméter vagy helytelen: (400 Bad Request)

{
	"error": "Grant type parameter missing or type not allowed!"
}

Hiányzik a felhasználónév paraméter: (400 Bad Request)

{
	"error": "Username parameter missing!"
}

Hiányzik a jelszó paraméter: (400 Bad Request)

{
	"error": "Password parameter missing!"
}

Helytelen felhasználónév: (400 Bad Request)

{
	"error": "No user with username: \"xxxxxxxxxx\"!"
}

Helytelen jelszó: (400 Bad Request)

{
	"error": "Incorrect password!"
}

Helytelen Content-Type: (400 Bad Request)

{
	"error": "Invalid Content-Type value or header missing!"
}

Helytelen body: (400 Bad Request)

{
	"error": "Invalid data in body!"
}

Helytelen URL: (404 Not Found)

{
	"error": "Page not found!"
}
