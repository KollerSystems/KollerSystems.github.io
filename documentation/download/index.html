Koller szerver letöltése

Automatikus (Ajánlott)

  1. Az automatikus telepítéssel a legegyszerűbb letölteni a szervert, hiszen minden szükséges fájlt automatikusan és gyorsan telepít biztonságos webhelyről. Az egyetlen egy hátránya, hogy szükséges dotnet 8.0-nak a gépen lennie. Ezt innen tudjuk telepíteni: https://dotnet.microsoft.com/en-us/download/dotnet/8.0

  2. Ha sikeresen letöltöttük és feltelepítettük a dotnet 8.0-t a számítógépünkre, töltsük le az automatikus szerver telepítőt a következő weboldalról: https://github.com/4E-6F-72-62-65-72-74/Koller-Installer/releases/latest

  3. Ha a telepítő letöltött, indítsuk el a kívánt mappában ahova a szervert szeretnénk telepíteni és mindent elvégez magától abban a mappában. Egy kivétel van, az adatbázis root jelszava. Itt magunknak be kell írni milyen jelszót szeretnénk és a telepítő elvégzi a feladatokat magától. Ha üresen hagyjuk, a telepítő magától generál egy jelszót. (A jelszó később kinézhető az options.json fájlból)

  4. Ha ezzel is megvagyunk, a telepítő létrehoz egy start.bat fájlt és egy update.bat fájlt. Az elsővel tudjuk elindítani a szervert, a másodikkal frissíteni tudjuk azt naprakészre. Indítsuk el a start.bat-ot és a szerver már fut is.

Manuális

  1. Töltsük le és telepítsük a NodeJS-t: https://nodejs.org/en/download/

  2. Töltsük le és telepítsük a Git-et: https://git-scm.com/download/

  3. Töltsük le és telepítsük a MariaDB-t: https://mariadb.org/download/

  4. Nyissunk egy parancssort és navigáljuk a számunkra tökéletes mappába, ahova a szervert fogjuk telepíteni, majd töltsük le GitHub-ról a szervert:

    git clone https://github.com/4E-6F-72-62-65-72-74/Koller-Server.git

    Ha ez készen van, töltsük le az adatbázishoz sükséges fájlokat is:

    git clone https://github.com/4E-6F-72-62-65-72-74/Koller-Database.git

  5. Indítsunk el egy MySQL klienst és futtassunk 2+1 parancsot:

    Adatbázis séma feltöltése:

    SOURCE .../Koller-Database/schema.sql;

    Adatbázis konfiguráció feltöltése:

    SOURCE .../Koller-Database/config.sql;

    Dummy felhasználók feltöltése: (opcionális)

    SOURCE .../Koller-Database/dummy.sql;

  6. Parancssorral navigáljunk a Koller-Server mappába és töltsük le a NodeJS modulokat a következő parancs segítségével:

    npm install

  7. A szerver sikeresen telepítve van, ha nincsen egy hiba sem. Innentől a szerver elindítható ezzel a parancs segítségével:

    npm start

    vagy clusterelés nélkül: (tesztelésre ajánlott)

    node index.js

Opcionális: Csináljuk indító batch fájlt a könnyebb indítás érdekében a fő mappában. Mentsük el a fájlt start.bat fájlként (Csak Windows)

@echo off
cd Koller-Server
node index.js
PAUSE

Opcionális: Csináljuk frissítő batch fájlt a könnyebb frissítés érdekében a fő mappában. Mentsük el a fájlt update.bat fájlként (Csak Windows)

@echo off
cd Koller-Server
git pull
cd ../Koller-Database
git pull
PAUSE
